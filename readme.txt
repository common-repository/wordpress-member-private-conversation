Simple Member Messaging System for WordPress Site.


== Description ==
NMedia Wordpress Member Conversation Plugin allow members to send and receive messages on any WordPress site. 
Lightning fast fronend UI. Setup in 5 seconds.

== Setup ==
* Create a page
* paste this shortocde: [nmconvo]
* Admin Settings: Dashboard -> WP Convo


== Features ==
* Ajax based conversation
* Users List Modal
* New conversation notification
* All conversations list in admin
* Multi Language supported
* Sanitization applied
* Responsive Design

[Demo & Detail](https://najeebmedia.com/wordpress-plugin/wordpress-front-end-member-private-message-with-file-attachment/)

== Pro Features ==
* Control communication by roles
* File Attachment
* File Type, Size Control
* User list sorting, searching, pagination by DataTabe API
* Lifetime updates
* Pro Support Access
* Email Notification Extended Control
* Email Message Customization


== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can set options from `WP Convo` menu
